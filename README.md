# MLOps Customer Analytics Platform

An end-to-end machine learning platform for customer churn prediction,
built to demonstrate production-style ML engineering practices:
experiment tracking, model registry, API serving, and containerized deployment.

## Status
🚧 Under active development (Week 1 of 6)

## Tech Stack
- **ML**: Python, Pandas, Scikit-learn, XGBoost
- **Experiment Tracking**: MLflow
- **Backend**: FastAPI
- **Frontend**: Streamlit
- **Database**: PostgreSQL
- **Deployment**: Docker
